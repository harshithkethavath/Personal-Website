Prompt Engineering Failure Modes and LoRA Adaptation in CLIPSeg for Satellite Cloud Segmentation

A controlled study of 60 prompt variants and fine-tuning strategies applied to CLIPSeg on the CloudSEN12+ dataset. Rather than chasing benchmark numbers, this work asks a harder question: where exactly do vision-language models break down, and can low-rank adaptation fix it systematically?

Key findings

Supervised fine-tuning outperforms prompt engineering reliably above an 8-image threshold — below that, prompting is competitive. We also identified a class-level supervision dip in spectrally ambiguous classes, where adding more labeled data temporarily degrades performance before recovering. This dip phenomenon is the paper's primary contribution.

MarsHazardNet: Multi-task Terrain Hazard Prediction on Mars CTX Imagery

A multi-task U-Net architecture for simultaneous hazard classification and terrain segmentation on Mars Context Camera imagery. The core question is whether shared representations across tasks improve reliability in low-data extraterrestrial environments.

WeatherMind: Multi-model VLM Evaluation for Sky Imagery Weather Analysis

A comparative framework for evaluating vision-language models — Qwen2-VL, SmolVLM2, Moondream2, Gemma3 — on weather analysis from ground-level sky photographs. Built to understand where each model's visual reasoning holds up and where it doesn't.
